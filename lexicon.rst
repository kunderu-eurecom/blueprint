
Lexicon
=======


* **CPRI:** Common Public Radio Interface
* **CU:** Central Unit
* **CU-CP:** Central Unit Control Plane
* **CU-UP:** Central Unit User Plane
* **DU:** Distributed Unit 
* **NREN:** National Research and Education Network
* **PTP:** Precision Time Protocol
* **RAN:** Radio Access Network
* **RU:**\ : Radio Unit
* **UE:** User Equipment
